## Sense-Aid

Sense-Aid is a wearable head mounted device, built to assist blind people navigate the world with ease. It consists of a camera, whose live feed is fed to an AI model, which generates an audio and streams it a predefined set of earphones worn by the user. This audio data given is descriptive and helps the user to navigate the world with ease.

<hr>

### Functionality

- Descriptive audio format to help the user navigate the world
- Gemini Live running via internet
- Haptic feedback via motors
